CSS II

Universal Selector & All property

The Star ( * ) selects each element. Super useful. Example:

  html {margin:value;}. //won't do.
  
  * {margin:0;} // will do.
  
  element {all:value;}

Resets all properties.

Universal selector selects elements, all selects properties.

Pseudoelements

Essentials

Syntaxis selector::PE {code}

Example h1::after {content: 'url(/images/xxx)'}

Specificity same than elements.

And they are...

  • first-line, first-letter
  • before, after
  • selection

Single colon also works, for cc1 & 2 compatibility.

before,after insert content before/after an element's content.

selection applies to the selection made by a user. Can be applied to color, background, cursor, outline.

Pseudoclass

Essentials

Syntaxis selector:PC {code}

Example p:first-child {code}

Specificity same than classes.

And they are...

  • :focus useful for input text or area.
  • :hover :visited :active
  • :first-child :last-child :nth-child
  • p:last-of-type (all elements last ps on any parent)
  • :not(selector) selects all that is not selector !

p { display: none; background-color: yellow; padding: 20px; }

div:hover p { background-color:red; }

When the div is hovered, it sets the display property of the paragraph

Position

Syntaxys selector{position:value;}

Values relative, absolute, fixed, sticky, static.

Offset top, bottom, right and left offsets.

Common Values

  • Static: default.
  • Relative: to the position in the normal flow. ~ removes the element from normal flow (element not ignored)
  • Absolute: respect to body or first relative positioned parent. Removes element from normal flow.
  • Fixed: is absolute positioning relative to the browser window!
  • Sticky: not sure.
  • Float: right or left.

Normal flow is the arrange of elements without using position abs or float.

Fixed v Absolute

Difference: fixed elements stay fixed to the window when scrolling.

When setting float:x or position:absolute, \n the rest of the elements ignore these elements.

More on Floats

It makes the floated element boyar around other element. It's very useful for images and sidebars.

Flex and Grids are interesting variants to flots with a nicer behaviour and properties.

Issues

Floats are absolute positioned i.e `recognized' height is $0$ and do not affect other elements' position. If inside a container, container's height will be zero.\footnote{try: write a background color. It will stay the same.}

Solutions

To the parent:

  • overflow:hidden; (automatically measures childs' height).

To neighboring elements:

  • If we want overflow visible and not to invade other elements, use clear:both;

How to center

  • Use position absolute in element, and position relative in the parent element.
  • Use flexbox.

Center Text

  • Flexbox or
  • Line height + align center.

z-Index

This is relevant when using position or background property (cause it allows to stack elements). By default, the element coming last on the html document is put on the top.

By giving z-index values we can alter this 'z-stack order'.
