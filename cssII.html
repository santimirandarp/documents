Brief

Universal Selector & All property

The Star ( * ) selects each elements' property. all selects a property from one element. Used for resets. Example: Universal selector selects elements, all selects properties.

Combinators

  • descendant body p
  • general sibling p ~ p (siblings: under same parent)
  • adjacent sibling li + li (li that is after an li).
  • direct children ul>li

Pseudoelements

Essentials

Syntaxis selector::PE {code}

Example h1::after {content: 'url(/images/xxx)'}

Specificity same than elements.

And they are...

  • first-line, first-letter
  • before, after
  • selection

Single colon also works, for cc1 & 2 compatibility.

before,after insert content before/after an element's content.

selection applies to the selection made by a user. Can be applied to color, background, cursor, outline.

Pseudoclass

Essentials

Syntaxis selector:PC {code}

Example p:first-child {code}

Specificity same than classes.

  • :focus useful for input text or area.
  • :hover :visited :active
  • :first-child :last-child :nth-child
  • p:last-of-type (all elements last ps on any parent)
  • :not(selector) selects all that is not selector !
p {
    display: none;
    background-color: yellow;
    padding: 20px;
    }
  div:hover p {
    background-color:red;
    }

When the div is hovered, it sets the display property of the paragraph.
