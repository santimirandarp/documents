Types of Network

Although boring, knowing the types of network simplifies things. We can classify networks depending on the area they span, and whether they are wired or wireless.

  • PAN, Personal Area Network. Devices in a PAN can be connected wireless via bluetooth, IR, NFS; or they can connect by cable.
  • Local Area Network or LAN, frequent on an office, or a house. Devices connect either to a Switch by Ethernet Cable or to a Router by a Wifi card. LAN is wired-only; WLAN is wireless instead. The Router connects to the Switch and this creates a closed network between wired and wireless devices.

    • Nowadays, the Router may include the switch. Same thing goes for CAN (campus area network, usually being wireless).

    MAN, WAN. Metropolitan or Worldwide Area network. These days the connection is mostly by fiber cables. The Internet, or World Wide Web is a WAN. Devices in this network are connected using fiber cables. We reach the WAN from the LAN if the Modem is connected to the WAN.

    We can run nice LANs without any connection to the WAN.

    WLAN

    Router means 'the one who draws the route'. This route is the path to a set of devices. Components: An antenna/boost, Modem, Router, Ethernet Switches. WAN or internet port where the Modem connects.

    Router

    WLAN requires login/authentication. Either the Wifi Router or the Switch assigns each device an IP. This is called DHCP. To transfer data the router needs port and ip address of the other computer/device. The port is specific of the application, 8080 for webs, 3636 for windows remote desktop (WRD) etc etc.

    Allows to create different networks

    Modem

    Modem, Router and Switch may be compressed into one object. They still have very different role.

    The modem is usually connected to a Fiber Cable or to a Satellite on one end. On the other end it connects to the Router or Switch.

    The word modem comes from modulate/demodulate. It converts the signal from computer-friendly to antenna friendly and viceversa. The modem connects to the ISP (internet service provider). The connection to the ISP might be Fiber or Cable.

    Details

    In large areas we want to extend the Router wifi-range, using WAP or Booster/wifi extender. WAP and Booster do not have switches, do not assign IPs to devices in the network (the Router does) an do not have a WAN port. So these connect to the router, using ethernet cables (like phone cables, 8 lines on the plug instead).

    Firewall

    A firewall config may not be necessary because this is done in the router.

    A firewall stops the fire, in this case, dangerous data. Host-based firewalls are the ones set up on a computer, and protect only that computer. Windows comes with firewall and also 3rd parties can be installed. Antivirus might come with firewall. Network-based firewall rules are applied to the entire network. This is probably set in the switch or Router, by tweaking its config using the PC.
