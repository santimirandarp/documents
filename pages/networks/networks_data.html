Types of Network

Although boring, knowing the types of network simplifies things. We can classify networks depending on the area they span, and whether they are wired or wireless.

  • pan. Devices in a PAN can be connected wireless via bluetooth, ir, nfs; or they can connect by cable.
  • lan. Frequent on an office, or a house. Devices connect either to a Switch by Ethernet Cable or to a Router by a Wifi card. lan is wired-only; wlan is wireless instead. The Router connects to the Switch and this creates a closed network between wired and wireless devices.

    • Nowadays, the Router may include the switch. Same thing goes for can (campus area network, usually being wireless).

    man, wan. Metropolitan or Worldwide Area network. These days the connection is mostly by fiber cables. The Internet, or World Wide Web is a wan. Devices in this network are connected using fiber cables. We reach the wan from the lan if the Modem is connected to the wan.

    We can run nice lans without any connection to the wan.

    Hardware on a Network

    • Modem: for DSL, Satellite, Cable, Fiber. Connects (data) our network with the external world (internet).
    • Router: the one who draws the route. This route is the path the data follows, from a device to a device. It connects different networks.
    • Firewall
    • Switch: connects cabled devices to a network. There are unmanaged and managed switches. The latter ones can be programmed for specific tasks. The switch supports particular speeds.
    • Wireless Access Point or WAP (optional): connects wifi devices to a network. It's linked to the switch.
    • Computer

    Modem

    The modem is usually connected to a Fiber Cable or to a Satellite on one end. On the other end it connects to the Router or Switch.

    The word modem comes from modulate/demodulate. It converts the signal from computer-friendly to antenna friendly and viceversa. The modem connects to the ISP (internet service provider). The connection to the ISP might be Fiber or Cable.

    WAP speed

    Wireless network devices (prefix=802.11). Examples: prefix-a, b, g, n. Most of them are compatible to each other (only a is useless), so buying any network card should work. We should pick up the best speed match though. n cover a good area and are quick.

    Firewall

    A firewall config may not be necessary because this is done in the router.

    A firewall stops the fire, in this case, dangerous data. Host-based firewalls are the ones set up on a computer, and protect only that computer. Windows comes with firewall and also 3rd parties can be installed. Antivirus might come with firewall. Network-based firewall rules are applied to the entire network. This is probably set in the switch or Router, by tweaking its config using the PC.

    Cabling

    Small Office, Home Office

    soho networks are small networks; like a lan at home. For this kind of networks we can buy a particular type of Routers. Enterprise routers only route traffic from the wan, to the lan. soho routers can include many devices apart from the router. Usually a Switch (those phone plugs on the back), a DNS server, DHCP server, Firewall server, and Port forwarding.

    We can configure a soho Router using a web browser, by typing the router's IP in the URL bar. This is not the way to set up an enterprise router.
