Basics of Floats

List of useful default values:

  1. overflow:visible
  2. float:none
  3. parent>children(floated). Children are 'pasted in' after document's render. They will overflow. Examples: image inside div, lis inside ul. We need to set up a few tricks. They produce nice effects though.

We only need clear on elements after the floats, the elements before the floats are not affected.

Source order matters

A float only affects elements coming after it.

For example, this paragraph is written after the image element. Floats change the behavior of siblings coming after it (and only after it).

So the first paragraph behaves as a block element and this ones wrap around.

Summary (wrapping around!)

Floated elements lead subsequent text and sibling elements to wrap them around.
