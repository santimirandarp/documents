Display Property & Floats

Display

Blocks can be thought as boxes. Always start on a new line and extends to the full available width.
To center a block-element horizontally respect to its parent container add: margin:auto; Note: it has no effect if the width is not set (or is set to 100%).

Inline-blocks don't start on a new line (as the name indicates) and do not extend to the full width but occupy minimum amount required. Width and height can be set for inline block elements. To center an image set 

    .centerImage{margin: auto; display:block;}
They behave as blocks in the sense that either wraps completely or do not wrap at all.

Inline elements do not start on a new line and take up as much width as necessary. There are many ways to center a line vertically: line height for one line, padding for one or multiple lines, position relative, and so on. Only width and line-height can be tweak through css, not height. To just center the text inside an element, use text-align: center;

<img> and <a> are inline elements, but the state can be modified with display property.

More values for display

Another value for display is none. And a similar one is visibility:hidden (but the space remains).

Floats

As the word says a floated element floats on its parent element. We tell this through CSS by using float:right|left|none*|inherit;. As indicated by the * float: none; is the default value.
