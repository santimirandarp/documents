HTML: Hypertext Markup Language

From Wikipedia:

Hypertext Markup Language [is] a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.

As different from previous versions, HTML5 includes semantic elements: tags with a clear meaning.

Some of those elements are: nav, main, section, article. This helps

  • programmers to understand the code,
  • screen readers to produce meaning.
